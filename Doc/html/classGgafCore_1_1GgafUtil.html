簡易ハッシュ .

文字列 を、さも一意のような32biｔ数値に変換。 スピード優先のハッシュ関数のため、厳密ではない。 33を乗じると、衝突が少ないらしいという文献を信じる。

引数:
str文字列
戻り値:
ハッシュ値

GgafUtil.h の 94 行で定義されています。

                                                   {
        UINT32 hash = 5381;
        char c;
        while ((c = *str++) > 0) { //strの\0までループ （演算子 "==" と間違えていません）
            hash = ((hash << 5) + hash) + c; // hash * 33 + c  33倍してます
        }
        return hash;
    }
